# Deployment Guide

## Gyoza Client Application

### Build Setup

install dependencies
```bash
$ yarn install
````

serve with hot reload at localhost:3000
```bash
$ yarn dev
````

build for production and launch server
```bash
$ yarn build
$ yarn start
````

generate static project
```bash
$ yarn generate
```
